# TP_CultureNumerique
Télecharger algobox
Télecharger la dernière version du fichier
Ouvrir algobox
En haut à gauche cliquez sur ouvrir, puis aller dans téléchargemet, séléctionnez le fichier et cliquer sur ouvrir
En haut à gauche cliquez sur Tester puis lancer le programme
Il vous propse déterminer les solutions d'une équation du second degré du type a*x^2+b*x+c=0
Entrez les valeur de a, b et c de votre équation
Le programme vous affiche une ou deux solutions réel ou imaginaire selon la valeur du discriminat(Delta)
